# GenshinCheat
Simple cheat for genshin with protection bypass.

## Features
- [X] Mhyprot bypass
- [X] Check integrity bypass
- [X] Ingame gui
- [X] Map teleportation
- [X] Teleport to chest and oculi
- [X] Infinite sprint with "no send move sync" mode
- [X] God mode, dumb enemies, rapid fire, auto talk, no cd
